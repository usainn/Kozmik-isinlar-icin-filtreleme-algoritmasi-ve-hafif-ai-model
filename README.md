# Kozmik-isinlar-icin-filtreleme-algoritmasi-ve-hafif-ai-model
Bilimsel çalışmalara destek sağlamak için başlangıç adımı olarak atılmış algoritma + model projesidir. Belirtilen Görsellerdeki ve beklendiği gibi filtreleme adımları başarıyla gerçekleşmiştir.



# Cosmic Ray Detection with Isolation Forest 

Bu proje, astronomik görüntülerdeki (FITS formatı) kozmik ışın parazitlerini temizlemek için Klasik Medyan Filtresi ve Isolation Forest (AI) yöntemlerini birleştirir.

## Özellikler
- Hafif Mimari: Karmaşık derin öğrenme modelleri yerine hızlı anomali tespiti.
- Astronomy Ready: Astropy kütüphanesi ile tam uyumlu.
- Görsel Analiz: Tespit edilen ışınların ısı haritası üzerinden takibi.

## Nasıl Çalıştırılır?
1. `pip install -r requirements.txt`
2. `python src/main.py`


## Neler Gerekli?
- numpy
- matplotlib
- astropy
- scipy
- scikit-learn
